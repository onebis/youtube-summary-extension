# Chrome Web Store Listing (English)

## Short Description

Summarize YouTube videos from subtitles with your own AI key. Supports Claude / OpenAI / Gemini. Output in Japanese or English.

Length: 125 / 132

## Detailed Description

YouTube Summary is a Chrome extension that fetches captions from YouTube videos and generates summaries using your own AI API key. The summary appears in the browser side panel so you can review it alongside the video.

Key features:

- Adds a Summarize button to YouTube video pages
- Summarizes video content from captions
- Outputs summaries in Japanese or English
- Supports Short / Medium / Detailed summary modes
- Shows summaries in the Chrome side panel
- Supports copy, regenerate, and cancel actions
- Supports Claude, OpenAI, and Gemini
- Uses a BYOK model: bring your own API key

Supported providers / models:

- Claude: `claude-sonnet-4-6` (recommended), `claude-opus-4-7`, `claude-haiku-4-5-20251001`
- OpenAI: `gpt-4o` (recommended), `gpt-4o-mini`, `gpt-4-turbo`
- Gemini: `gemini-2.5-flash` (recommended), `gemini-2.5-pro`, `gemini-2.0-flash`

Privacy:

- The developer does not collect user data
- API keys are stored only in local browser storage
- Only when summarization is started, the caption text, video ID, and output language are sent to the AI provider selected by the user
- No external error logging, cookies, or tracking are used

How to use:

1. Open the extension options page and configure your AI provider and API key.
2. Open a YouTube video page.
3. Click the Summarize button on the video page.
4. Review the generated summary in the side panel.

Notes:

- Videos without captions cannot be summarized.
- AI API usage costs and limits depend on each provider account.
- This extension is not an official product of YouTube or any AI provider.
