YouTube動画を、
AI で要約。

字幕から構造化された要約を生成する Chrome 拡張機能。あなた自身の AI API キーを使う BYOK 方式。アカウント不要、トラッキングなし、開発者サーバーを経由しません。

GitHub で見る Chrome Web Store 公開準備中

できること

字幕がある YouTube 動画なら、すべて構造化された AI 要約に。

構造化された出力

見出しと箇条書きで整理された要約。読みにくい文章の壁ではありません。

日英バイリンガル

動画の言語に関わらず、日本語または英語で要約を生成できます。

サイドパネル表示

ページを離れず、動画の横で要約を読めます。

3 つの要約モード

ざっくり知りたい時の Short、文脈も追いたい時の Medium、全部知りたい時の Detailed。

キーもデータもあなたのもの

API キーはブラウザ内に保存されるのみ。分析・テレメトリ・中継サーバーは一切なし。

コピー・再生成・中止

ワンクリックでコピー、同じ入力で再実行、長いリクエストはいつでも中止可能。

実際の動作画面

構造化された要約が Chrome のサイドパネルに表示されます。別タブを開かず、動画の横で読めます。

  1. 1 日英バイリンガル要約の出力言語を日本語と英語でワンタップ切替。
  2. 2 3 つの要約モードShort (ざっくり) / Medium (標準) / Detailed (詳細) を切り替え。
  3. 3 構造化された出力カテゴリタグ・見出し・箇条書きで整理。文章の壁にはなりません。

使い方

一度設定したら、あとは押すだけ。

  1. API キーを設定

    オプション画面で Claude / OpenAI / Gemini を選び、API キーを貼り付けます。ブラウザにのみ保存されます。

  2. YouTube 動画を開く

    動画ページのアクションバー (高評価などの並び) に Summarize ボタンが追加されます。

  3. Summarize をクリック

    Chrome のサイドパネルが開き、構造化された要約が表示されます。

データは、あなたのもの

BYOK (Bring Your Own Key) 方式のため、開発者を一切経由しません。

  • ユーザー登録不要、開発者のサーバーを経由しません
  • API キーはブラウザのローカルストレージにのみ保存
  • 字幕は、あなたが選択した AI プロバイダにのみ送信
  • 分析・テレメトリ・サードパーティトラッキングは一切なし
