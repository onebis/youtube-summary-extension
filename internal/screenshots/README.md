# Screenshot Assets

Chrome Web Store screenshots must be captured manually from a clean Chrome profile after loading `dist/` as an unpacked extension.

Required files:

- `01.png`: YouTube video page with the Summarize button visible
- `02.png`: Side panel while loading
- `03.png`: Side panel with a Japanese summary
- `04.png`: Side panel with an English summary
- `05.png`: Options page with provider selection

Use `1280x800` or `640x400` PNG/JPG files.
