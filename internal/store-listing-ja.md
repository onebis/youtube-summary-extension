# Chrome Web Store リスティング（日本語）

## 短い説明

YouTube動画を字幕からAI要約。Claude/OpenAI/Gemini対応、BYOK方式で安心。日本語・英語で要約可能。

文字数: 61 / 132

## 詳細説明

YouTube動画要約は、YouTube動画の字幕を取得し、ユーザー自身のAI APIキーで要約を生成するChrome拡張機能です。動画を見ながらサイドパネルで要約を確認できます。

主な機能:

- YouTube動画ページに「Summarize」ボタンを追加
- 字幕をもとに動画内容をAIで要約
- 日本語または英語で要約を出力
- 短め / 中間 / 詳細の要約モードを切り替え
- サイドパネルで動画を見ながら要約を確認
- 要約結果のコピー、再生成、中止に対応
- Claude / OpenAI / Gemini からプロバイダを選択
- APIキーはユーザー自身で用意するBYOK方式

対応プロバイダ / モデル:

- Claude: `claude-sonnet-4-6`（推奨）、`claude-opus-4-7`、`claude-haiku-4-5-20251001`
- OpenAI: `gpt-4o`（推奨）、`gpt-4o-mini`、`gpt-4-turbo`
- Gemini: `gemini-2.5-flash`（推奨）、`gemini-2.5-pro`、`gemini-2.0-flash`

プライバシー:

- 開発者はユーザーデータを収集しません
- APIキーはブラウザ内のローカルストレージにのみ保存されます
- 要約実行時のみ、字幕本文、動画ID、出力言語がユーザー選択のAIプロバイダへ送信されます
- エラーログ、Cookie、トラッキングは使用しません

使い方:

1. 拡張機能の設定画面でAIプロバイダとAPIキーを設定します。
2. YouTube動画ページを開きます。
3. 動画ページ上の「Summarize」ボタンを押します。
4. サイドパネルで要約を確認します。

注意:

- 字幕がない動画は要約できません。
- AI APIの利用料金や利用制限は、各プロバイダのアカウント設定に従います。
- この拡張機能はYouTubeまたは各AIプロバイダの公式製品ではありません。
